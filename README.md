WordPowerMadeEasy

## Installation
To install this app 
* To get regular updates we recommend you to install [f-droid.org](http://fdroid.org), a free app-store for open source free apps. Click here to start [FDroid.apk](https://f-droid.org/FDroid.apk) download.
* Find and install
  _[Word Power Made Easy](https://f-droid.org/repository/browse/?fdid=com.developfreedom.wordpowermadeeasy)_ from the fdroid app store
* You're done :)
